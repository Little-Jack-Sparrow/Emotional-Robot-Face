# Emotional-Robot-Face 情感机器人面部表情系统
电子科技大学 电子信息工程专业 本科在读，专注于仿生机器人面部表情驱动、多模态情感交互、二次元IP具身化落地的学习与项目开发。

## 项目介绍
本仓库记录我从零基础开始，学习机器人面部表情相关技术的完整成长轨迹，核心目标是实现「二次元IP的具身情感陪伴机器人」，覆盖从算法到硬件落地的全链路技术。

## 核心技术栈
编程语言：Python、C/C++
视觉工具：OpenCV
深度学习框架：PyTorch
硬件开发：STM32、嵌入式系统开发
核心方向：人脸关键点检测、表情识别与生成、语音驱动唇形同步(Lip Sync)、多模态情感对齐、机器人面部舵机驱动
